### 🚀 Selected Research Projects

#### 1. [TRB/TSL] Sequential Investment Decision via Transformer-PPO
> **Focus:** Sequential Network Expansion & Real Option Analysis
- **Method:** Proposed **TPPO (Transformer-based PPO)** with Dual-head Decoding & Residual Feature Fusion.
- **Highlight:** Integrated **LSMC** for reward shaping to solve sparse feedback in long-horizon decisions.
- **Outcome:** Outperformed Transformer-SAC; NPV improved by **34.7%** in NYC experiments.
- 🔗 [Code & Paper](./) #### 2. [RLHF] Autonomous Fleet Integration in On-Demand Fulfillment
> **Focus:** Human-AI Alignment & Safety Compliance
- **Method:** **RLHF (Reinforcement Learning from Human Feedback)** pipeline (SFT -> Reward Model -> PPO).
- **Highlight:** Aligned autonomous fleet behavior with human dispatcher preferences using the Bradley-Terry model.
- **Outcome:** Reduced human takeover rate by **35%** in corner cases.
- 🔗 [Code & Paper](./)

#### 3. [EJOR] Multi-Objective Order Fulfillment with Order Postponement
> **Focus:** High-dimensional Multi-objective Trade-off
- **Method:** Graph-based Multi-Objective RL with Pareto Set Learning (**GMORL-PSL**).
- **Highlight:** Coupled MOMDP with MOVRPTW; utilized Hypernetwork for dynamic policy generation.
- **Outcome:** Reduced inter-period workload imbalance by **21.43%**.
- 🔗 [Code & Paper](./)

#### 4. [IJPR] Sustainable Pharmaceutical Logistics Routing
> **Focus:** Vehicle Flow Equilibrium & Cold Chain
- **Method:** Mixed-Integer Linear Programming (**MILP**) & Hybrid Heuristics (**SAVNS**).
- **Highlight:** Explicitly modeled vehicle flow balance constraints for collaborative depots.
- **Outcome:** Published in **IJPR (JCR Q1)**. Achieved **98.13%** loading rate.
